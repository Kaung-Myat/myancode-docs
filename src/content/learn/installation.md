# Installation

MyanCode runs anywhere Node.js runs. You can install the command-line tool globally, or skip installation entirely and use the **Web IDE**.

## Option 1 — npm (recommended)

Install the CLI globally with npm:

```bash
npm install -g myancode
```

Verify the install:

```bash
myancode --version
```

## Option 2 — Web IDE (no install)

Don't want to install anything? Open the browser editor and start writing immediately:

> [myan-code.vercel.app](https://myan-code.vercel.app)

The Web IDE ships with example programs, a Run button, and the same engine as the CLI — fully offline once loaded.

## Option 3 — From source

For contributors who want the latest code:

```bash
git clone https://github.com/Kaung-Myat/Myan-Code
npm install
npm start
```

## Requirements

- **Node.js v18+** for the CLI and source builds
- **Any modern browser** for the Web IDE

Once installed, head to **[Hello World](/learn/hello-world)** to write your first program.
