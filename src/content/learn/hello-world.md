# Hello World

Every programming journey starts with a greeting. In MyanCode, you print to the screen with the keyword **`ပြောပါ`** (*"say"*).

Create a file named `hello.myan`:

```myancode
    ပြောပါ "မင်္ဂလာပါ ကမ္ဘာ"
```

That's a complete program. The `ပြောပါ` keyword takes whatever follows it — here a text string — and prints it.

## Printing in English too

Strings can hold any text, Burmese or English:

```myancode
    ပြောပါ "Hello, World"
    ပြောပါ "မြန်မာကုဒ်မှ ကြိုဆိုပါတယ်"
```

Each `ပြောပါ` statement prints on its own line, so the output is:

```
    Hello, World
    မြန်မာကုဒ်မှ ကြိုဆိုပါတယ်
```

## Printing values

You can also print numbers and variables directly — no quotes needed for numbers:

```myancode
    ပြောပါ ၂၀၂၆
```

> MyanCode reads Burmese digits (`၀–၉`) natively. `၂၀၂၆` is the number **2026**.
